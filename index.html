How to Create an OpenRouter API Key

Follow these steps to generate an API key for OpenRouter:

  1. Go to the OpenRouter website.
  2. Sign in or create an account if you donâ€™t have one.
  3. Navigate to the API Keys section in your account settings.
  4. Click on Create Key.
  5. Click on create and create your API key.
  6. Copy and securely store your API key. Do not share it publicly.
  7. Add the API key to the chrome extention form.
Note: API keys are sensitive credentials. Always keep them private and use them securely in your applications.
